# n2n 多平台自动编译

自动从 [ntop/n2n](https://github.com/ntop/n2n) 上游拉取 Release，编译并发布多平台预编译二进制文件。

### 平台支持
| 平台    | 架构   | 文件 |
|---------|--------|------|
| Linux   | x86    | `n2n-3.0-linux-x86.tar.gz`      |
| Linux   | amd64  | `n2n-3.0-linux-amd64.tar.gz`    |
| Linux   | arm64  | `n2n-3.0-linux-arm64.tar.gz`    |
| Windows | amd64  | `n2n-3.0-windows-amd64.zip`     |
| macOS   | arm64 (M1+) | `n2n-3.0-macos-arm64.tar.gz` |
